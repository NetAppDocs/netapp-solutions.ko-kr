StorageGRIDлҠ” л¶„мӮ°лҗң л…ёл“ң кё°л°ҳ к·ёлҰ¬л“ң м•„нӮӨн…ҚмІҳлҘј мӮ¬мҡ©н•ҳм—¬ м •мұ… мӨ‘мӢ¬мқҳ м§ҖлҠҘнҳ• кёҖлЎңлІҢ лҚ°мқҙн„° кҙҖлҰ¬лҘј м ңкіөн•©лӢҲлӢӨ. м •көҗн•ң лҚ°мқҙн„° кҙҖлҰ¬ кё°лҠҘкіј кІ°н•©лҗң мң л№„мҝјн„°мҠӨ кёҖлЎңлІҢ мҳӨлёҢм қнҠё л„Өмһ„мҠӨнҺҳмқҙмҠӨлҘј нҶөн•ҙ нҺҳнғҖл°”мқҙнҠё лӢЁмң„мқҳ л№„м •нҳ• лҚ°мқҙн„°мҷҖ мҲҳмӢӯм–ө к°ңмқҳ мҳӨлёҢм қнҠё кҙҖлҰ¬лҘј к°„мҶҢнҷ”н•©лӢҲлӢӨ. лӢЁмқј нҳём¶ң к°ңмІҙ м•Ўм„ёмҠӨлҠ” мӮ¬мқҙнҠё к°„м—җ нҷ•мһҘлҗҳкі  кі к°Җмҡ©м„ұ м•„нӮӨн…ҚмІҳлҘј лӢЁмҲңнҷ”н•ҳлҠ” лҸҷмӢңм—җ мӮ¬мқҙнҠё лҳҗлҠ” мқён”„лқј мӨ‘лӢЁкіј кҙҖкі„м—Ҷмқҙ м§ҖмҶҚм Ғмқё к°ңмІҙ м•Ўм„ёмҠӨлҘј ліҙмһҘн•©лӢҲлӢӨ.

л©ҖнӢ° н…Ңл„ҢмӢңлҘј мӮ¬мҡ©н•ҳл©ҙ лҸҷмқјн•ң к·ёлҰ¬л“ң лӮҙм—җм„ң м—¬лҹ¬ л№„м •нҳ• нҒҙлқјмҡ°л“ң л°Ҹ м—”н„°н”„лқјмқҙмҰҲ лҚ°мқҙн„° м• н”ҢлҰ¬мјҖмқҙм…ҳмқ„ м•Ҳм „н•ҳкІҢ м„ңл№„мҠӨн•  мҲҳ мһҲмңјлҜҖлЎң NetApp StorageGRIDмқҳ ROI л°Ҹ мӮ¬мҡ© мӮ¬лЎҖк°Җ мҰқк°Җн•©лӢҲлӢӨ. м—¬лҹ¬ м§Җм—ӯм—җм„ң лӮҙкө¬м„ұ, ліҙнҳё, м„ұлҠҘ, мқём ‘м„ұмқ„ мөңм Ғнҷ”н•ҳм—¬ л©”нғҖлҚ°мқҙн„° кё°л°ҳ мҳӨлёҢм қнҠё лқјмқҙн”„мӮ¬мқҙнҒҙ м •мұ…мқ„ нҶөн•ҙ м—¬лҹ¬ м„ңл№„мҠӨ л ҲлІЁмқ„ мғқм„ұн•  мҲҳ мһҲмҠөлӢҲлӢӨ. мӮ¬мҡ©мһҗлҠ” лҚ°мқҙн„° кҙҖлҰ¬ м •мұ…мқ„ мЎ°м •н•ҳкі  нҠёлһҳн”Ҫ м ңн•ңмқ„ лӘЁлӢҲн„°л§Ғ л°Ҹ м Ғмҡ©н•ҳм—¬ лҒҠмһ„м—Ҷмқҙ ліҖнҷ”н•ҳлҠ” IT нҷҳкІҪм—җм„ң мҡ”кө¬ мӮ¬н•ӯмқҙ ліҖкІҪлҗЁм—җ л”°лқј лҚ°мқҙн„° нҷҳкІҪмқ„ мӨ‘лӢЁ м—Ҷмқҙ лӢӨмӢң мЎ°м •н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

Grid ManagerлҘј нҶөн•ң к°„нҺён•ң кҙҖлҰ¬

StorageGRID к·ёлҰ¬л“ң кҙҖлҰ¬мһҗлҠ” лёҢлқјмҡ°м Җ кё°л°ҳмқҳ к·ёлһҳн”Ҫ мқён„°нҺҳмқҙмҠӨлЎң, лӢЁмқј м°Ҫм—җм„ң м „м„ёкі„м—җ л¶„мӮ°лҗң мң„м№ҳм—җ кұёміҗ StorageGRID мӢңмҠӨн…ңмқ„ кө¬м„ұ, кҙҖлҰ¬ л°Ҹ лӘЁлӢҲн„°л§Ғн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

StorageGRID к·ёлҰ¬л“ң кҙҖлҰ¬мһҗ мқён„°нҺҳмқҙмҠӨлҘј мӮ¬мҡ©н•ҳм—¬ лӢӨмқҢ мһ‘м—…мқ„ мҲҳн–үн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

  • мқҙлҜём§Җ, л№„л””мҳӨ, л ҲмҪ”л“ң л“ұ м „ м„ёкі„м—җ л¶„мӮ°лҗң нҺҳнғҖл°”мқҙнҠё к·ңлӘЁмқҳ мҳӨлёҢм қнҠё м ҖмһҘмҶҢлҘј кҙҖлҰ¬н•©лӢҲлӢӨ.

  • к·ёлҰ¬л“ң л…ёл“ң л°Ҹ м„ңл№„мҠӨлҘј лӘЁлӢҲн„°л§Ғн•ҳм—¬ к°ңмІҙ к°Җмҡ©м„ұмқ„ ліҙмһҘн•©лӢҲлӢӨ.

  • ILM(м •ліҙ мҲҳлӘ… мЈјкё° кҙҖлҰ¬) к·ңм№ҷмқ„ мӮ¬мҡ©н•ҳм—¬ мӢңк°„мқҙ м§ҖлӮЁм—җ л”°лқј мҳӨлёҢм қнҠё лҚ°мқҙн„°мқҳ л°°м№ҳлҘј кҙҖлҰ¬н•©лӢҲлӢӨ. мқҙлҹ¬н•ң к·ңм№ҷмқҖ мҲҳм§‘лҗң к°ңмІҙмқҳ лҚ°мқҙн„°, лҚ°мқҙн„°к°Җ мҶҗмӢӨлҗҳм§Җ м•ҠлҸ„лЎқ ліҙнҳён•ҳлҠ” л°©лІ•, мҳӨлёҢм қнҠё лҚ°мқҙн„°к°Җ м ҖмһҘлҗҳлҠ” мң„м№ҳ л°Ҹ кё°к°„м—җ лҢҖн•ҙ м Ғмҡ©лҗ©лӢҲлӢӨ.

  • мӢңмҠӨн…ң лӮҙмқҳ нҠёлһңмһӯм…ҳ, м„ұлҠҘ л°Ҹ мҡҙмҳҒмқ„ лӘЁлӢҲн„°л§Ғн•©лӢҲлӢӨ.

м •ліҙ мҲҳлӘ… мЈјкё° кҙҖлҰ¬ м •мұ…

StorageGRIDлҠ” мҳӨлёҢм қнҠёмқҳ ліөм ңліё ліҙмЎҙ л°Ҹ нҠ№м • м„ұлҠҘ л°Ҹ лҚ°мқҙн„° ліҙнҳё мҡ”кө¬мӮ¬н•ӯм—җ л”°лқј мҳӨлёҢм қнҠёлҘј м ҖмһҘн•  мҲҳ мһҲлҸ„лЎқ 2+1 л°Ҹ 4+2(нҠ№нһҲ лӢӨлҘё)мҷҖ к°ҷмқҖ EC(мӮӯм ң мҪ”л”©) мҠӨнӮӨл§ҲлҘј мӮ¬мҡ©н•ҳлҠ” л“ұмқҳ мң м—°н•ң лҚ°мқҙн„° кҙҖлҰ¬ м •мұ…мқ„ м ңкіөн•©лӢҲлӢӨ. мӢңк°„м—җ л”°лқј мӣҢнҒ¬лЎңл“ңмҷҖ мҡ”кө¬мӮ¬н•ӯмқҙ лӢ¬лқјм§ҖлӮ мҲҳлЎқ ILM м •мұ…лҸ„ мӢңк°„м—җ л”°лқј л°”лҖҢм–ҙм•ј н•©лӢҲлӢӨ. ILM м •мұ…мқ„ мҲҳм •н•ҳлҠ” кІғмқҖ н•өмӢ¬ кё°лҠҘмқҙлҜҖлЎң StorageGRID кі к°қмқҖ лҒҠмһ„м—Ҷмқҙ ліҖнҷ”н•ҳлҠ” нҷҳкІҪм—җ л№ лҘҙкі  мүҪкІҢ м Ғмқ‘н•  мҲҳ мһҲмҠөлӢҲлӢӨ. лҘј нҷ•мқён•ҳмӢӯмӢңмҳӨ "ILM м •мұ…" л°Ҹ StorageGRIDм—җм„ң м„Өм •.

м„ұлҠҘ

StorageGRIDлҠ” VM, лІ м–ҙ л©”нғҲ лҳҗлҠ” нҠ№мҲҳ м ңмһ‘лҗң м–ҙн”Ңлқјмқҙм–ёмҠӨ л“ұ мҠӨнҶ лҰ¬м§Җ л…ёл“ңлҘј м¶”к°Җн•ҳм—¬ м„ұлҠҘмқ„ нҷ•мһҘн•©лӢҲлӢӨ "SG5712, SG5760, SG6060 лҳҗлҠ” SGF6024". мқҙ н…ҢмҠӨнҠём—җм„ңлҠ” SGF6024 м–ҙн”Ңлқјмқҙм–ёмҠӨлҘј мӮ¬мҡ©н•ҳлҠ” мөңмҶҢ нҒ¬кё°мқҳ 3л…ёл“ң к·ёлҰ¬л“ңлЎң Apache Kafkaмқҳ мЈјмҡ” м„ұлҠҘ мҡ”кө¬мӮ¬н•ӯмқ„ мҙҲкіјн–ҲмҠөлӢҲлӢӨ. кі к°қмқҙ м¶”к°Җ лёҢлЎңм»ӨлЎң Kafka нҒҙлҹ¬мҠӨн„°лҘј нҷ•мһҘн•Ём—җ л”°лқј мҠӨнҶ лҰ¬м§Җ л…ёл“ңлҘј м¶”к°Җн•ҳм—¬ м„ұлҠҘкіј мҡ©лҹүмқ„ нҷ•мһҘн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

л¶Җн•ҳ л¶„мӮ° мһҘм№ҳ л°Ҹ м—”л“ңнҸ¬мқёнҠё кө¬м„ұ

StorageGRIDмқҳ кҙҖлҰ¬ л…ёл“ңлҠ” StorageGRID мӢңмҠӨн…ңмқ„ ліҙкі , кө¬м„ұн•ҳкі , кҙҖлҰ¬н•  мҲҳ мһҲлҠ” к·ёлҰ¬л“ң кҙҖлҰ¬мһҗ UI(мӮ¬мҡ©мһҗ мқён„°нҺҳмқҙмҠӨ) л°Ҹ REST API м—”л“ңнҸ¬мқёнҠёмҷҖ мӢңмҠӨн…ң мһ‘м—…мқ„ м¶”м Ғн•  мҲҳ мһҲлҠ” к°җмӮ¬ лЎңк·ёлҘј м ңкіөн•©лӢҲлӢӨ. Confluent Kafka кі„мёөнҳ• мҠӨнҶ лҰ¬м§Җм—җ к°Җмҡ©м„ұмқҙ лҶ’мқҖ S3 м—”л“ңнҸ¬мқёнҠёлҘј м ңкіөн•ҳкё° мң„н•ҙ кҙҖлҰ¬ л…ёл“ңмҷҖ кІҢмқҙнҠёмӣЁмқҙ л…ёл“ңм—җм„ң м„ңл№„мҠӨлЎң мӢӨн–үлҗҳлҠ” StorageGRID лЎңл“ң л°ёлҹ°м„ңлҘј кө¬нҳ„н–ҲмҠөлӢҲлӢӨ. лҳҗн•ң лЎңл“ң л°ёлҹ°м„ңлҠ” лЎңм»¬ нҠёлһҳн”Ҫмқ„ кҙҖлҰ¬н•ҳкі  GSLB(Global Server Load Balancing)м—җ м—°кІ°н•ҳм—¬ мһ¬н•ҙ ліөкө¬лҘј м§Җмӣҗн•©лӢҲлӢӨ.

м—”л“ңнҸ¬мқёнҠё кө¬м„ұмқ„ лҚ”мҡұ к°ңм„ н•ҳкё° мң„н•ҙ StorageGRIDлҠ” кҙҖлҰ¬ л…ёл“ңм—җ лӮҙмһҘлҗң нҠёлһҳн”Ҫ л¶„лҘҳ м •мұ…мқ„ м ңкіөн•ҳкі , мӣҢнҒ¬лЎңл“ң нҠёлһҳн”Ҫмқ„ лӘЁлӢҲн„°л§Ғн•ҳкі , мӣҢнҒ¬лЎңл“ңм—җ лӢӨм–‘н•ң QoS(м„ңл№„мҠӨ н’Ҳм§Ҳ) м ңн•ңмқ„ м Ғмҡ©н•  мҲҳ мһҲлҸ„лЎқ м§Җмӣҗн•©лӢҲлӢӨ. нҠёлһҳн”Ҫ л¶„лҘҳ м •мұ…мқҖ кІҢмқҙнҠёмӣЁмқҙ л…ёл“ң л°Ҹ кҙҖлҰ¬ л…ёл“ңм—җ лҢҖн•ң StorageGRID л¶Җн•ҳ л¶„мӮ° м„ңл№„мҠӨмқҳ лҒқм җм—җ м Ғмҡ©лҗ©лӢҲлӢӨ. мқҙлҹ¬н•ң м •мұ…мқҖ нҠёлһҳн”Ҫ м ңн•ң л°Ҹ лӘЁлӢҲн„°л§Ғмқ„ м§Җмӣҗн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

StorageGRIDмқҳ нҠёлһҳн”Ҫ л¶„лҘҳ

StorageGRIDм—җлҠ” QoS кё°лҠҘмқҙ лӮҙмһҘлҗҳм–ҙ мһҲмҠөлӢҲлӢӨ. нҠёлһҳн”Ҫ л¶„лҘҳ м •мұ…мқҖ нҒҙлқјмқҙм–ёнҠё м• н”ҢлҰ¬мјҖмқҙм…ҳм—җм„ң л“Өм–ҙмҳӨлҠ” лӢӨм–‘н•ң мң нҳ•мқҳ S3 нҠёлһҳн”Ҫмқ„ лӘЁлӢҲн„°л§Ғн•ҳлҠ” лҚ° лҸ„мӣҖмқҙ лҗ  мҲҳ мһҲмҠөлӢҲлӢӨ. к·ёлҹ° лӢӨмқҢ м •мұ…мқ„ мғқм„ұн•ҳм—¬ м Ғмҡ©н•ҳм—¬ In/Out лҢҖм—ӯнҸӯ, мқҪкё°/м“°кё° лҸҷмӢң мҡ”мІӯ мҲҳ лҳҗлҠ” мқҪкё°/м“°кё° мҡ”мІӯ мҶҚлҸ„м—җ л”°лқј мқҙ нҠёлһҳн”Ҫм—җ м ңн•ңмқ„ м Ғмҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

м•„нҢҢм№ҳ м№ҙн”„м№ҙ

Apache KafkaлҠ” Java л°Ҹ ScalaлЎң мһ‘м„ұлҗң мҠӨнҠёлҰј мІҳлҰ¬лҘј мӮ¬мҡ©н•ҳлҠ” мҶҢн”„нҠёмӣЁм–ҙ лІ„мҠӨмқҳ н”„л Ҳмһ„мӣҢнҒ¬мһ…лӢҲлӢӨ. мқҙ м ңн’ҲмқҖ мӢӨмӢңк°„ лҚ°мқҙн„° н”јл“ң мІҳлҰ¬лҘј мң„н•ң лҶ’мқҖ мІҳлҰ¬лҹүмқҳ м§§мқҖ лҢҖкё° мӢңк°„мқ„ к°–м¶ҳ нҶөн•© н”Ңлһ«нҸјмқ„ м ңкіөн•ҳкё° мң„н•ҙ л§Ҳл Ёлҗҳм—ҲмҠөлӢҲлӢӨ. KafkaлҠ” лҚ°мқҙн„° лӮҙліҙлӮҙкё° л°Ҹ к°Җм ёмҳӨкё°лҘј мң„н•ҙ Kafka ConnectлҘј нҶөн•ҙ мҷёл¶Җ мӢңмҠӨн…ңм—җ м—°кІ°н•  мҲҳ мһҲмңјл©° Java мҠӨнҠёлҰј мІҳлҰ¬ лқјмқҙлёҢлҹ¬лҰ¬мқё Kafka мҠӨнҠёлҰјмқ„ м ңкіөн•©лӢҲлӢӨ. KafkaлҠ” нҡЁмңЁм„ұмқ„ мң„н•ҙ мөңм Ғнҷ”лҗң л°”мқҙл„ҲлҰ¬ TCP кё°л°ҳ н”„лЎңнҶ мҪңмқ„ мӮ¬мҡ©н•ҳл©°, л„ӨнҠёмӣҢнҒ¬лҘј нҶөн•ң мҷ•ліө мһ‘м—…мқҳ мҳӨлІ„н—Өл“ңлҘј мӨ„мқҙкё° мң„н•ҙ мһҗм—°мҠӨлҹҪкІҢ л©”мӢңм§ҖлҘј к·ёлЈ№нҷ”н•ҳлҠ” "л©”мӢңм§Җ м„ёнҠё" м¶”мғҒнҷ”м—җ мқҳмЎҙн•©лӢҲлӢӨ. мқҙл ҮкІҢ н•ҳл©ҙ мҲңм°Ё л””мҠӨнҒ¬ мһ‘м—…, лҚ” нҒ° л„ӨнҠёмӣҢнҒ¬ нҢЁнӮ· л°Ҹ м—°мҶҚ л©”лӘЁлҰ¬ лё”лЎқмқҙ мҰқк°Җн•ҳлҜҖлЎң KafkaлҠ” лһңлҚӨ л©”мӢңм§Җ м“°кё°мқҳ нҸӯмЈј мҠӨнҠёлҰјмқ„ м„ нҳ• м“°кё°лЎң м „нҷҳн•  мҲҳ мһҲмҠөлӢҲлӢӨ. лӢӨмқҢ к·ёлҰјмқҖ Apache Kafkaмқҳ кё°ліё лҚ°мқҙн„° нқҗлҰ„мқ„ ліҙм—¬ мӨҚлӢҲлӢӨ.

KafkaлҠ” мғқмӮ°мһҗлқјлҠ” мһ„мқҳмқҳ мҲҳмқҳ н”„лЎңм„ёмҠӨм—җм„ң к°Җм ёмҳЁ нӮӨ к°’ л©”мӢңм§ҖлҘј м ҖмһҘн•©лӢҲлӢӨ. лҚ°мқҙн„°лҠ” м—¬лҹ¬ мЈјм ң лӮҙмқҳ м„ңлЎң лӢӨлҘё нҢҢнӢ°м…ҳмңјлЎң л¶„н• лҗ  мҲҳ мһҲмҠөлӢҲлӢӨ. нҢҢнӢ°м…ҳ лӮҙм—җм„ң л©”мӢңм§ҖлҠ” н•ҙлӢ№ мҳӨн”„м…Ӣ(нҢҢнӢ°м…ҳ лӮҙмқҳ л©”мӢңм§Җ мң„м№ҳ)м—җ мқҳн•ҙ м—„кІ©н•ҳкІҢ м •л ¬лҗҳкі  нғҖмһ„ мҠӨнғ¬н”„мҷҖ н•Ёк»ҳ мқёлҚұмӢұлҗҳкі  м ҖмһҘлҗ©лӢҲлӢӨ. мҶҢл№„мһҗлқјкі  н•ҳлҠ” лӢӨлҘё н”„лЎңм„ёмҠӨлҠ” нҢҢнӢ°м…ҳм—җм„ң л©”мӢңм§ҖлҘј мқҪмқ„ мҲҳ мһҲмҠөлӢҲлӢӨ. мҠӨнҠёлҰј мІҳлҰ¬лҘј мң„н•ҙ KafkaлҠ” Kafkaмқҳ лҚ°мқҙн„°лҘј мӮ¬мҡ©н•ҳлҠ” Java м• н”ҢлҰ¬мјҖмқҙм…ҳмқ„ мһ‘м„ұн•ҳкі  кІ°кіјлҘј Kafkaм—җ лӢӨмӢң м“ё мҲҳ мһҲлҠ” мҠӨнҠёлҰј APIлҘј м ңкіөн•©лӢҲлӢӨ. Apache KafkaлҠ” Apache Apex, Apache Flink, Apache Spark, Apache Storm, Apache nifi л“ұмқҳ мҷёл¶Җ мҠӨнҠёлҰј мІҳлҰ¬ мӢңмҠӨн…ңкіјлҸ„ мһ‘лҸҷн•©лӢҲлӢӨ.

KafkaлҠ” н•ҳлӮҳ мқҙмғҒмқҳ м„ңлІ„(лёҢлЎңм»Ө)лЎң кө¬м„ұлҗң нҒҙлҹ¬мҠӨн„°м—җм„ң мӢӨн–үлҗҳл©° лӘЁл“  н•ӯлӘ©мқҳ нҢҢнӢ°м…ҳмқҙ нҒҙлҹ¬мҠӨн„° л…ёл“ңм—җ л¶„мӮ°лҗ©лӢҲлӢӨ. лҳҗн•ң нҢҢнӢ°м…ҳмқҙ м—¬лҹ¬ лёҢлЎңм»Өм—җ ліөм ңлҗ©лӢҲлӢӨ. мқҙ м•„нӮӨн…ҚмІҳлҘј нҶөн•ҙ KafkaлҠ” лӮҙкІ°н•Ём„ұмқҙ мһҲлҠ” л°©мӢқмңјлЎң лҢҖк·ңлӘЁ л©”мӢңм§Җ мҠӨнҠёлҰјмқ„ м „лӢ¬н•  мҲҳ мһҲмңјл©° JMS(Java Message Service), AMQP(Advanced Message Queuing Protocol) л“ұмқҳ кё°мЎҙ л©”мӢңм§• мӢңмҠӨн…ңмқ„ лҢҖмІҙн•  мҲҳ мһҲмҠөлӢҲлӢӨ. KafkaлҠ” 0.11.0.0 лҰҙлҰ¬мҠӨ мқҙнӣ„ нҠёлһңмһӯм…ҳ м“°кё°лҘј м ңкіөн•ҳм—¬ мҠӨнҠёлҰј APIлҘј мӮ¬мҡ©н•ҳм—¬ м •нҷ•нһҲ н•ң лІҲмқҳ мҠӨнҠёлҰј мІҳлҰ¬лҘј м ңкіөн•©лӢҲлӢӨ.

KafkaлҠ” мқјл°ҳкіј м»ҙнҢ©м…ҳмқҙлқјлҠ” л‘җ к°Җм§Җ мң нҳ•мқҳ мЈјм ңлҘј м§Җмӣҗн•©лӢҲлӢӨ. м •к·ң мЈјм ңлҠ” ліҙмЎҙ мӢңк°„ лҳҗлҠ” кіөк°„ л°”мқёл”©мңјлЎң кө¬м„ұн•  мҲҳ мһҲмҠөлӢҲлӢӨ. м§Җм •лҗң ліҙмЎҙ мӢңк°„ліҙлӢӨ мҳӨлһҳлҗң л ҲмҪ”л“ңк°Җ мһҲкұ°лӮҳ нҢҢнӢ°м…ҳм—җ лҢҖн•ҙ л°”мқёл”©лҗң кіөк°„мқҙ мҙҲкіјлҗң кІҪмҡ° KafkaлҠ” мӮ¬мҡ© к°ҖлҠҘн•ң м ҖмһҘмҶҢ кіөк°„мқ„ нҷ•ліҙн•ҳкё° мң„н•ҙ мқҙм „ лҚ°мқҙн„°лҘј мӮӯм ңн•  мҲҳ мһҲмҠөлӢҲлӢӨ. кё°ліём ҒмңјлЎң н•ӯлӘ©мқҖ ліҙмЎҙ кё°к°„мқҙ 7мқјлЎң кө¬м„ұлҗҳм§Җл§Ң лҚ°мқҙн„°лҘј л¬ҙкё°н•ң м ҖмһҘн•  мҲҳлҸ„ мһҲмҠөлӢҲлӢӨ. м••м¶•лҗң н•ӯлӘ©мқҳ кІҪмҡ° мӢңк°„ лҳҗлҠ” кіөк°„ лІ”мң„м—җ л”°лқј л ҲмҪ”л“ңк°Җ л§ҢлЈҢлҗҳм§Җ м•ҠмҠөлӢҲлӢӨ. лҢҖмӢ  KafkaлҠ” лӮҳмӨ‘м—җ л°ӣмқҖ л©”мӢңм§ҖлҘј лҸҷмқјн•ң нӮӨлҘј мӮ¬мҡ©н•ҳлҠ” мқҙм „ л©”мӢңм§Җмқҳ м—…лҚ°мқҙнҠёлЎң м·Ёкёүн•ҳл©° нӮӨлӢ№ мөңмӢ  л©”мӢңм§ҖлҘј мӮӯм ңн•ҳм§Җ м•ҠлҸ„лЎқ ліҙмһҘн•©лӢҲлӢӨ. мӮ¬мҡ©мһҗлҠ” нҠ№м • нӮӨм—җ лҢҖн•ҙ null к°’мқ„ к°–лҠ” мҶҢмң„ tombstone л©”мӢңм§ҖлҘј мһ‘м„ұн•ҳм—¬ л©”мӢңм§ҖлҘј мҷ„м „нһҲ мӮӯм ңн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

Kafkaм—җлҠ” лӢӨмқҢкіј к°ҷмқҖ 5к°Җм§Җ мЈјмҡ” APIк°Җ мһҲмҠөлӢҲлӢӨ.

  • * Producer API. * мқ‘мҡ© н”„лЎңк·ёлһЁм—җм„ң л ҲмҪ”л“ң мҠӨнҠёлҰјмқ„ кІҢмӢңн•  мҲҳ мһҲлҸ„лЎқ н—Ҳмҡ©н•©лӢҲлӢӨ.

  • * мҶҢл№„мһҗ API. * мқ‘мҡ© н”„лЎңк·ёлһЁм—җм„ң н•ӯлӘ© л°Ҹ л ҲмҪ”л“ң мҠӨнҠёлҰј н”„лЎңм„ёмҠӨлҘј кө¬лҸ…н•  мҲҳ мһҲлҸ„лЎқ н—Ҳмҡ©н•©лӢҲлӢӨ.

  • * Connector API. * н•ӯлӘ©мқ„ кё°мЎҙ мқ‘мҡ© н”„лЎңк·ёлһЁм—җ м—°кІ°н•  мҲҳ мһҲлҠ” мһ¬мӮ¬мҡ© к°ҖлҠҘн•ң н”„лЎңл“Җм„ң л°Ҹ мҶҢл№„мһҗ APIлҘј мӢӨн–үн•©лӢҲлӢӨ.

  • * Streams API. * мқҙ APIлҠ” мһ…л Ҙ мҠӨнҠёлҰјмқ„ м¶ңл ҘмңјлЎң ліҖнҷҳн•ҳкі  кІ°кіјлҘј мғқм„ұн•©лӢҲлӢӨ.

мҶҢл№„мһҗ л°Ҹ мғқмӮ°мһҗ APIлҠ” Kafka л©”мӢңм§• н”„лЎңнҶ мҪңмқ„ кё°л°ҳмңјлЎң н•ҳл©° Javaмқҳ Kafka мҶҢл№„мһҗ л°Ҹ мғқмӮ°мһҗ нҒҙлқјмқҙм–ёнҠём—җ лҢҖн•ң м°ёмЎ° кө¬нҳ„мқ„ м ңкіөн•©лӢҲлӢӨ. кё°ліё л©”мӢңм§• н”„лЎңнҶ мҪңмқҖ к°ңл°ңмһҗк°Җ н”„лЎңк·ёлһҳл°Қ м–ём–ҙлЎң мҶҢл№„мһҗ лҳҗлҠ” мғқмӮ°мһҗ нҒҙлқјмқҙм–ёнҠёлҘј мһ‘м„ұн•ҳлҠ” лҚ° мӮ¬мҡ©н•  мҲҳ мһҲлҠ” мқҙм§„ н”„лЎңнҶ мҪңмһ…лӢҲлӢӨ. мқҙл ҮкІҢ н•ҳл©ҙ JVM(Java Virtual Machine) м—җмҪ”мӢңмҠӨн…ңм—җм„ң Kafkaк°Җ мһ кёҲ н•ҙм ңлҗ©лӢҲлӢӨ. мӮ¬мҡ© к°ҖлҠҘн•ң л№„ Java нҒҙлқјмқҙм–ёнҠё лӘ©лЎқмқҖ Apache Kafka wikiм—җм„ң мң м§Җ кҙҖлҰ¬лҗ©лӢҲлӢӨ.

Apache Kafka мӮ¬мҡ© мӮ¬лЎҖ

Apache KafkaлҠ” л©”мӢңм§•, мӣ№ мӮ¬мқҙнҠё нҷңлҸҷ м¶”м Ғ, л©”нҠёлҰӯ, лЎңк·ё м§‘кі„, мҠӨнҠёлҰј мІҳлҰ¬, мқҙлІӨнҠё мҶҢмӢұ л°Ҹ лЎңк№… м»Өл°Ӣ

  • KafkaлҠ” н–ҘмғҒлҗң мІҳлҰ¬лҹү, лӮҙмһҘ нҢҢнӢ°м…”лӢқ, ліөм ң л°Ҹ лӮҙкІ°н•Ём„ұ кё°лҠҘмқ„ м ңкіөн•ҳлҜҖлЎң лҢҖк·ңлӘЁ л©”мӢңм§Җ мІҳлҰ¬ м• н”ҢлҰ¬мјҖмқҙм…ҳм—җ м Ғн•©н•ң мҶ”лЈЁм…ҳмһ…лӢҲлӢӨ.

  • KafkaлҠ” мӢӨмӢңк°„ кІҢмӢң кө¬лҸ… н”јл“ң м§‘н•©мңјлЎң м¶”м Ғ нҢҢмқҙн”„лқјмқём—җм„ң мӮ¬мҡ©мһҗмқҳ нҷңлҸҷ(нҺҳмқҙм§Җ ліҙкё°, кІҖмғү)мқ„ мһ¬кө¬м¶•н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

  • KafkaлҠ” мҡҙмҳҒ лӘЁлӢҲн„°л§Ғ лҚ°мқҙн„°м—җ мһҗмЈј мӮ¬мҡ©лҗ©лӢҲлӢӨ. мқҙлҘј мң„н•ҙм„ңлҠ” л¶„мӮ°лҗң м• н”ҢлҰ¬мјҖмқҙм…ҳм—җм„ң нҶөкі„лҘј м§‘кі„н•ҳм—¬ мҡҙмҳҒ лҚ°мқҙн„°мқҳ мӨ‘м•ҷ м§‘мӨ‘мӢқ н”јл“ңлҘј мғқм„ұн•ҳлҠ” мһ‘м—…мқҙ н•„мҡ”н•©лӢҲлӢӨ.
